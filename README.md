# Quick installation

## Local Run

Download

```bash
wget https://github.com/whale-music/whale/releases/latest/download/whale-music.jar
```

### run

```bash
java -jar whale-music.jar
```

# Build

> Not including the web

clone

```bash
git clone https://github.com/whale-music/whale
cd whale
```
maven build
```bash
mvn clean package -DskipTests
```
